# GraphQlNetCoreAspNet
GraphQL bound to a .Net Core, Asp.Net controller. Locking tribal knowledge in a minimalistic project.

# Playground
Playground is available at: http://localhost:5427/ui/playground

# Example query
```
query {
  foo(category: "bar") {
    id,
    category,
    friendlyName,
    date
  }
}
```
